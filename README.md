# ShiroVerseAPI

[![JitPack](https://img.shields.io/jitpack/v/github/tantaihaha4487/ShiroVerseAPI.svg)](https://jitpack.io/#tantaihaha4487/ShiroVerseAPI)

Add JitPack repository:

**Maven**
```
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
```
<dependency>
            <groupId>com.github.tantaihaha4487</groupId>
            <artifactId>ShiroVerseAPI</artifactId>
            <version>1.21-SNAPSHOT</version>
</dependency>
```

**Gradle (Groovy)**
```
repositories { maven { url 'https://jitpack.io' } }
```
```
dependencies { implementation 'com.github.tantaihaha4487:ShiroVerseAPI:1.21-SNAPSHOT' }
```

**Gradle (Kotlin)**
```
repositories { maven("https://jitpack.io") }
dependencies { implementation("com.github.tantaihaha4487:ShiroVerseAPI:1.21-SNAPSHOT") }
```
